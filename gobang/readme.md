# 五子棋游戏

基于PyQt5实现的简易五子棋游戏，支持双人对战和人机对战模式。

## 功能特性

### 游戏模式
- **双人对战模式**：同一设备上，两名玩家交替落子，完成对弈
- **人机对战模式**：玩家与AI进行对弈，AI支持基础的落子逻辑（优先堵截、优先成五）

### 棋盘功能
- 标准15×15路棋盘
- 清晰的网格线，黑白棋子对比明显
- 点击落子点即可落子，落子后不可移动或撤回
- 同一落子点不可重复落子
- 双人模式下，黑棋先手，黑白交替落子
- 人机模式下，玩家可选择先手/后手

### 胜负判定
- 任意一方在横、竖、斜向形成连续5颗同色棋子，立即判定该方胜利
- 棋盘填满且无胜负，判定为平局

### 辅助功能
- **规则说明**：展示标准五子棋规则
- **游戏重置**：对弈过程中可重新开始游戏，有确认弹窗避免误操作

## 技术实现

- 使用PyQt5框架构建GUI界面
- 采用面向对象的设计模式
- 实现了基础的AI落子逻辑

## 运行环境

- Python 3.6+
- PyQt5库

## 安装与运行

1. 安装PyQt5库：
   ```bash
   pip install PyQt5
   ```

2. 运行游戏：
   ```bash
   python gobang.py
   ```

## 界面截图

游戏主界面：
- 显示"双人对战"、"人机对战"、"规则说明"三个选项

棋盘界面：
- 15×15路棋盘，浅木色底色，深棕色网格线
- 界面顶部显示当前回合
- 底部有"重新开始"和"返回主菜单"按钮

## 交互说明

1. **落子**：点击棋盘上的落子点即可落子
2. **选择模式**：在主界面选择双人对战或人机对战
3. **选择先手**：人机对战时可选择执黑棋（先手）或白棋（后手）
4. **查看规则**：点击"规则说明"按钮查看详细规则
5. **重新开始**：对弈过程中点击"重新开始"按钮可重置游戏
6. **返回主菜单**：点击"返回主菜单"按钮可返回主界面

## 开发说明

### 主要文件
- `gobang.py`：游戏主程序文件，包含所有游戏逻辑和界面代码

### 主要类
- `GobangGame`：游戏主类，管理游戏状态和界面切换
- `BoardWidget`：棋盘组件，负责棋盘绘制和落子处理
